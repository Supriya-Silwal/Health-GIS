This mini project is focused on mapping and analyzing health facilities in Nepal using WebGIS technology. The goal is to provide an interactive platform for visualizing the spatial distribution of healthcare services at multiple administrative levels.

This application is built on a three-tier architecture:

  • Presentation tier: HTML, CSS, JavaScript with Leaflet.js for map interaction
  • Middleware tier: GeoServer for serving spatial layers (WMS/WFS)
  • Data tier: PostgreSQL/PostGIS for spatial data storage and queries
Tip: Hover for architecture diagram preview.

  • Base Maps: OpenStreetMap, ESRI Satellite, OpenTopoMap, CartoDB
  • Overlay Layers: Province boundaries, District boundaries, Local units, Health facility locations
Click any layer name for a visual example.
